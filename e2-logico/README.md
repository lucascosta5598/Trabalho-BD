# E2 - Modelo Lógico

Esta pasta contém os artefatos da segunda etapa do projeto, correspondente ao modelo lógico do banco de dados.

O modelo lógico foi elaborado a partir do DER desenvolvido na etapa anterior, realizando o mapeamento das entidades e relacionamentos para tabelas do modelo relacional.

Foram definidos:

* Tabelas do sistema;
* Chaves primárias (PK);
* Chaves estrangeiras (FK);
* Tipos de dados;
* Restrições de integridade.

Arquivos desta pasta:

* Esquema_Relacional.pdf
* Justificativa_Relacionamentos.pdf

O modelo lógico representa a estrutura que será utilizada posteriormente na implementação do banco de dados em PostgreSQL.
