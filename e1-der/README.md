# E1 - Modelo Conceitual (DER)

Esta etapa apresenta o modelo conceitual do sistema de academia.

Foram identificadas as entidades, atributos e relacionamentos necessários para representar as regras de negócio da academia.

Arquivos desta pasta:

* DER.pdf
* Decisoes_Modelagem.pdf

Principais entidades:

* Aluno
* Instrutor
* Plano
* Pagamento
* Treino
* Equipamento
* Aula Coletiva
* Frequência

Os relacionamentos foram definidos com suas respectivas cardinalidades e participações, seguindo os requisitos da disciplina.
