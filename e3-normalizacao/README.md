# E3 - Normalização

Esta pasta contém a análise das formas normais do banco de dados desenvolvido para o sistema de academia.

O objetivo desta etapa foi verificar a qualidade do esquema relacional, identificar possíveis redundâncias e garantir a integridade dos dados por meio da aplicação dos conceitos de normalização estudados na disciplina.

Foram analisadas as seguintes tabelas:

* ALUNO
* PAGAMENTO
* TREINO

Para cada tabela foram identificadas:

* Dependências funcionais;
* Primeira Forma Normal (1FN);
* Segunda Forma Normal (2FN);
* Terceira Forma Normal (3FN);
* Forma Normal de Boyce-Codd (BCNF).

Arquivo desta pasta:

* Normalizacao.pdf

A análise foi realizada com base nos conceitos apresentados por Elmasri & Navathe (7ª edição).
