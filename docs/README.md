
# Documentação do Projeto

Esta pasta reúne os documentos finais e complementares produzidos durante o desenvolvimento do projeto da disciplina de Banco de Dados.

O projeto consiste em um sistema de gerenciamento para academias, desenvolvido ao longo do semestre com base nos conceitos de modelagem de dados, normalização e implementação de bancos de dados relacionais.

## Conteúdo da Pasta

* Relatorio_Final.pdf
* Apresentacao_Final.pdf
* Dicionario_de_Dados.pdf
* Demais documentos complementares do projeto

## Objetivo

Centralizar toda a documentação produzida durante o desenvolvimento do sistema, servindo como material de consulta, apoio à apresentação e registro das decisões tomadas ao longo do projeto.
